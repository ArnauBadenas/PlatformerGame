## Delivery 01 - PlatformerGame

### Description
A 2D platformer where the player collects coins, uses powerups, and navigates obstacles to reach the end of each level.

### Features
 - Jumping
 - Powerups
 - Coins

### Controls
#### Keyboard
- Move: WASD  
- Jump: Space  
- Quit: Esc
